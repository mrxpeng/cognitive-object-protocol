# Why not Markdown?

Markdown remains useful. COP does not try to replace all Markdown files.

Markdown is weak when the document must support:

- block-level stable IDs;
- local model edits;
- structured human feedback;
- explicit trust and review states;
- evidence and counter-evidence links;
- graph relations;
- database persistence;
- context packet generation.

COP should export to Markdown for compatibility, but Markdown is not the canonical source for high-value AI-mediated knowledge workflows.
