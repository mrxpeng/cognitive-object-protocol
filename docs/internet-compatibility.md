# Internet Compatibility

COP is not a browser standard. It should integrate with the current internet through exports and adapters:

- HTML viewer;
- Markdown export;
- JSON API;
- GitHub repository storage;
- MCP server;
- DOCX/PDF export (roadmap item, not implemented in the v0.2 alpha CLI);
- Notion/Feishu/Slack adapters;
- SQL/Graph/Vector database adapters.

The core object remains COP; the display and transport layers are adapters.
