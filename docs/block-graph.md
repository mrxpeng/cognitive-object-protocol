# Block Graph

The block graph is the core COP structure.

## Blocks

Blocks are stable, addressable cognitive units. Examples:

- summary
- claim
- evidence
- risk
- decision
- task
- validation

## Relations

Relations describe how blocks connect:

- evidence supports claim;
- risk applies to claim;
- task resolves risk;
- validation checks task;
- comment targets claim.

This enables precise retrieval, local edits, and cross-document knowledge reuse.
