# Suggested Discussion Topics

1. Should the core unit be Block, Claim, or another primitive?
2. Is COP too complex compared with Markdown?
3. Should relations be required or optional?
4. Should operations be JSON Patch compatible?
5. Should COP prioritize human readability or model efficiency?
6. How should context packets be generated?
7. Should COP integrate with MCP?
8. What is the minimal useful block type set?
9. How should trust and review state be represented?
10. Should COP be a file format, a protocol, or both?
11. Should HTML be only a view layer or also a canonical representation?
12. How should domain-specific extensions be governed?
